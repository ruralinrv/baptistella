// Modelo de Real Options — Black-Scholes Adaptado para Ativos Reais

// Opção de Expansão (Call Option)
S₀ = VP dos fluxos futuros da expansão = R$ 186,4 mi
K = Custo da expansão (5.000 ha × R$ 48.000) = R$ 240,0 mi
σ = Volatilidade dos fluxos = 28%
T = Tempo para exercer = 5 anos
r = Taxa livre de risco = 10,5%

d₁ = [ln(186,4/240,0) + (0,105 + 0,28²/2) × 5] / (0,28 × √5) = 0,724
d₂ = 0,724 − 0,28 × √5 = 0,098
C = 186,4 × N(0,724) − 240,0 × e^(−0,105×5) × N(0,098)
C = R$ 82,4 mi

// Valor Total das Opções Reais
Opções = Expansão + Switch + Abandono = 82,4 + 34,6 + 18,2 = R$ 135,2 mi

// Valor pelo Método de Real Options
V (Real Options) = DCF Base (R$ 1.140 mi) + Opções (R$ 135,2 mi) = R$ 1.275 mi ≈ R$ 1,28 bi
🎲 Por que Real Options importa O DCF tradicional assume um caminho fixo — mas no agro, o produtor tem flexibilidade: pode expandir quando o preço sobe, trocar cultura quando o milho paga mais, ou abandonar talhões ruins. Essa flexibilidade tem valor real. Os R$ 135 mi de opcionalidade representam 10,6% do valor total. Ignorar isso subavalia o grupo.
